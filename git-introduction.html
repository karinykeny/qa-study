Versionamento de arquivo

Como criar um diretório local?

Resposta:

init

Como configurar o nome do usuário no git?

Resposta:

git config user.name nome

Como configurar o email de usuário no git?

Resposta:

git config user.email e-mail

Como limpar o terminal do git?

Resposta:

clear

Como verificar o status do repositório?

Resposta:

git status

Como adicionar um arquivo?

Resposta:

git add .

Como ver os commits?

Resposta:

git log

Como criar uma Branch?

Resposta:

git checkout -b nome da branch

Como consultar as branchs?

Resposta:

git branch

Como mudar de Branch?

Resposta:

git checkout nome da branch

Como apagar uma Branch local?

Resposta:

git branch -d nome da branch

Como fazer um merge entre branchs?

Resposta:

git merge nome da branch
